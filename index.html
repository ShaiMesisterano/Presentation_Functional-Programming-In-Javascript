Principles

Pure Function

  • Return value is only determined by its input values
  • Without observable side effects
  • Math.cos(x) will, for the same value of x, always return the same result. It doesn't:
    1. Change x
    2. Write to log files
    3. Do network requests
    4. Change program state
    5. etc.

Impure

var values = { a: 1 };
function impureFunction ( items ) {
  var b = 1;
  items.a = items.a * b + 2;
}
impureFunction( values );

Pure

var values = { a: 1 };
function pureFunction ( a ) {
  var b = 1;
  a = a * b + 2;
  return a;
}
values.a = pureFunction( values.a );

Higher Order Functions

Command vs. Query

// Command

var x = [14, 3, 77];
var y = x.splice(1, 2);
console.log(x); // [14]
console.log(y); // [3, 77]

// Query

var x = [14, 3, 77];
var y = x.slice(1, 3);
console.log(x); // [14, 3, 77]
console.log(y); // [3, 77]

Disadvantages

  1. Not all functionality is natively supported
  2. Not natural
  3. Harder to follow the flow
